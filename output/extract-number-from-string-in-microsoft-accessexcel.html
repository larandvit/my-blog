Extract Number from String in Microsoft Access/Excel

Date Sat 18 January 2020 Category Linux Tags microsoft / access / excel / regexp

Microsft Access and Excel don't include any string functions to extarct a number from a string. It should be created a custom function to complete this task. Regular Expressions are a good option to deal with manupulating text data. Microsoft Access and Excel are lacking of support of regular expressions but they alllow to utilize third party libriries. A library our interest is Microsoft VBScript Regular Expressions 5.5.

Open your Microsft Access database or Excel spreadsheet, then go to VBA Editor pressing Alt-F11 combitation
